# WundSTARK · Website-Entwürfe

Drei Design-Entwürfe für die neue Website des **WundSTARK-Wundzentrums** (Viersen & Mönchengladbach).
Alle drei bündeln dieselben Inhalte in je einer eigenständigen Startseite und unterscheiden sich in
Tonalität, Typografie und Layout.

## 🔗 Live ansehen

Über GitHub Pages: **https://36ilyas.github.io/wundstark/**

Die Startseite stellt die drei Entwürfe vor und verlinkt sie mit Live-Vorschau.

## Die Entwürfe

| # | Richtung | Charakter |
|---|----------|-----------|
| **1a** | Ruhig & vertrauensvoll · Serif-Display | Warmes Creme, Vollbild-Hero, Serifenschrift (Spectral) |
| **1b** | Klinisch-klar & professionell · Grotesk-Grid | Klare Raster, kräftige Grotesk-Headlines (Bricolage), viel Weißraum |
| **1c** | Warm & menschlich · Editorial | Sandtöne, weiche Rundungen, redaktioneller Rhythmus |

## Struktur

```
index.html        → Startseite / Übersicht der drei Entwürfe
entwurf-1a.html   → Entwurf 1a (Serif)
entwurf-1b.html   → Entwurf 1b (Grotesk-Grid)
entwurf-1c.html   → Entwurf 1c (Editorial)
source/           → Original-Design-Canvas (Quelle der Entwürfe)
```

## Hinweise

- Reine statische HTML-Seiten, keine Build-Schritte nötig. Zum lokalen Ansehen genügt ein
  einfacher Webserver, z. B. `python -m http.server` im Projektordner.
- Texte, Team-Namen und Bilder sind **Platzhalter** und können ausgetauscht werden.
- Die verwendeten Fotos stammen von [Unsplash](https://unsplash.com) und dienen nur zur Illustration.
